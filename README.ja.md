# Red Sprite Filter

高空での急瞬発光現象（TLE：Transient Luminous Event）の観測向け macOS デスクトップツールです。カメラで撮影した雷雲の映像から、レッドスプライト（赤色スプライト）の疑いがある稲妻のフレームとクリップを自動的にふるい分けます。

## できること

レッドスプライトは通常、短時間で、かすかに、高空に現れ、繊細で複雑な形状をしています。長い映像を手作業でフレームごとに探すのは非常に時間がかります。このツールは映像全体をスキャンし、各フレームにレッドスプライトの候補としてスコアを付け、次のものを自動的に書き出します。

- 候補のキーフレーム
- 候補の短いクリップ
- `candidates.csv`
- `contact_sheet.jpg`
- `report.html`
- 目視確認後の `confirmed_candidates.csv`

最初の候補を見つけた時点で止まることは**ありません**。映像全体をスキャンし、その中からスコアが最も高い候補イベントを選び出します。

## 画面プレビュー

現在のバージョンは macOS ネイティブのウィンドウアプリで、ローカル画面の表示に WebView を使用しています。ブラウザは開かず、映像の処理もネットワークにアップロードされることはありません。

画面の構成：

- 映像の選択 / フォルダの選択 / 出力先フォルダの選択
- 精密ふるい分け / 高再現ふるい分け
- 候補フレームのグリッド表示
- 候補クリップの再生
- スキャン進捗、経過時間、推定残り時間
- 元動画のタイムライン上でのクリップ時刻表示（例：`3:12-3:14`）
- スコア基準と高 / 中 / 低の疑わしい候補分類
- 確定 / 疑わしい / 除外 の目視確認
- レポートを開く / 結果フォルダを開く

## 初めて開くとき

現在のビルドは Apple Developer ID で署名・公証されていないため、macOS が「開発元を確認できません」と警告する場合があります。

開く手順：

1. `红色精灵筛选器.app` を右クリック
2. 「開く」を選択
3. ダイアログでもう一度「開く」をクリック

## 動作に必要な環境

現在の macOS アプリには `numpy` と `Pillow` が同梱されているため、Python パッケージを手動でインストールする必要はありません。

対象の Mac に引き続き必要なもの：

- `ffmpeg`
- `ffprobe`

Homebrew でのインストールを推奨：

```bash
brew install ffmpeg
```

アプリは起動時に依存関係を確認します。`ffmpeg not found` や `ffprobe not found` と表示された場合は、Homebrew で `ffmpeg` がインストールされていることを確認してください。

## 推奨する使い方

### 単一映像の精密ふるい分け

特定の映像にレッドスプライトが写っている可能性が高い場合に適しています。

推奨パラメータ：

- モード：精密
- 最大候補数：24
- 最小スコア：0.8
- 最小赤ピクセル数：8
- 前処理秒数：1.0 秒
- 後処理秒数：2.0 秒

### フォルダの一括事前ふるい分け

一連の雷雲素材を手早く確認したい場合に適しています。

推奨パラメータ：

- モード：高再現
- 最大候補数：24
- 最小スコア：0.012
- 最小赤ピクセル数：8

事前ふるい分けの後、怪しい映像に対して「精密ふるい分け」で個別に再実行してください。

## スコア基準と分類

候補は確認の優先度として次のように表示されます。

- `≥ 5.0`：高度に疑わしい候補
- `1.0 - 5.0`：中程度に疑わしい候補
- `< 1.0`：低い疑わしさの候補

この分類は確認の優先度を示す目安であり、最終判断ではありません。キーフレーム、書き出されたクリップ、元動画を合わせて確認してください。

## 候補がレッドスプライトかどうかの見分け方

レッドスプライトらしい候補は、次の特徴を備えていることが多いです。

- 雲頂の上、または画面の高い位置にある
- 赤または赤紫（マゼンタ）がはっきりしている
- 形状が局所的で、地平線全体が赤く光るのではない
- 縦の柱状、ひも状、クラゲ状、または細かい枝分かれの構造がある
- 持続時間が極めて短く、通常はわずか数フレームの間だけはっきりする

よくある誤検出（false positive）は次のとおりです。

- 通常の雲内雷や雲地雷
- 街灯、車のライト、塔の灯り、航空機の灯り
- 地平線の赤い光
- レンズフレア
- 大きな露出変化
- 固定されたホットピクセルや圧縮ノイズ

## プライバシー

すべての映像はお使いの端末内で処理されます。このツールが映像、フレーム、候補クリップ、パス情報をアップロードすることは一切ありません。

## 現在の状況

これは写真ワークフロー用ツールの初期公開バージョンです。ふるい分けを補助するためのものであり、人の目視確認に代わるものではありません。誤検出、見逃し、異なるカメラでの挙動、機能の提案など、Issue やフィードバックを歓迎します。

## 今後の予定

- 一括結果のサマリー画面を追加
- より多くの実サンプルでふるい分けのしきい値を調整
